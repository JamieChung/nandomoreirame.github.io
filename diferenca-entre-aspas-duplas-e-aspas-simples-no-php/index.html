Diferença entre aspas duplas e aspas simples no PHP

3/4/2012 | Deixe um comentário | PHP

Aspas duplas: fazem pre-processamento das váriaveis.

Ex:

<?php
  $var = "olá mundo";
  echo "Resposta: $var";
?>

Resultado:

Resposta: olá mundo

Aspas simples: não fazem pre-processamento das váriaveis. Entende tudo como uma String completa.

Ex:

<?php
  $var = "olá mundo";
  echo "Resposta: $var";
?>

Resultado:

Resposta: $var

Então? Otimize seu código, quando você estiver certo de que não ira ter váriaveis dentro das aspas use aspas simples, assim o PHP não irá ficar procurando váriaveis para pre-processar.
